# Trademark and Branding Guidance

DMARC Analyzer Azure is an open-source project name used in this repository.

## Allowed uses

- Describing compatibility or derivation in factual terms.
- Referencing this project in documentation, benchmarks, or comparisons.

## Restricted uses

- Implying endorsement by this project without permission.
- Reusing project branding in a way that can confuse users about source or support.
- Using logos, badges, or marks that suggest an official release when not official.

## Fork guidance

If you publish a fork:
- Clearly identify it as a fork.
- Use distinct release naming for your distributions.
- Do not present forked binaries as official project releases.
